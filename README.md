<h3 align="center">The New Order: First Days of Scanlines</h3>

  <p align="center">
Supports 4K resolution and below (4096 × 2160)<br><br>Ever thought, "Wow, my eyes aren't searing enough from this UI! The brain rot doesn't feel that bad!"? Well, here I have you scanlines on the entire screen for that true immersion. The filter is based on the TNO Terrain filter, and a savior for your eyes is that all other GUIs are not affected, only the map and the army screen for some reason.
  </p>

There is some flickering after you select and enter a country, unfortunately I can't fix that and just to play it safe,

## If you have epilepsy or any other negative condition that is created by flickering, please do not play this mod.

Some things to note, I created this without any DLCs and I haven't tested it out after starting the game as I just haven't felt like doing a TNO playthrough in a while, but there is absolutely nothing that should happen to break it. If there is, I have a [discord server](https://discord.gg/KXwx6bM9f9) for a mod I've been working on with a North America map centered around a US collapse if the Soviets and the US switched sides called Land of the Divided that you could join and tell me. Thanks!

## Usage

Download from [Steam Workshop]([https://steamcommunity.com/sharedfiles/filedetails/?id=3121138290](https://steamcommunity.com/sharedfiles/filedetails/?id=3122860605)), or download it and create a .mod file in the /mods directory of Hearts of Iron IV.
